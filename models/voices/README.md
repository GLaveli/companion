# Vozes clonadas (GPT-SoVITS)

Coloque aqui áudios de referência para clonar personagens.

```
models/voices/
  minha-personagem/
    ref.wav          # 3–10s de fala limpa
```

Depois registre o perfil no app (UI futura) ou veja `docs/VOICE.md`.

**Exemplo anime:** clip em japonês + `prompt_lang: ja` + `text_lang: pt` → Lotus fala pt-BR com timbre da personagem.
